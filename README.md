# Role Memory Forge

给 SillyTavern 用的角色扮演记忆插件：自动把每轮 `{{user}}` 和角色回复压缩成简记，按层级生成阶段总结和大总结，并实时维护角色状态、人物档案、人物关系、世界设定、物品栏、约定、当前剧情与发展方向。

## 0.2.0 新增

- 新增右下角悬浮球，点击后打开完整记忆控制台。
- 设置全部移入精美浮层面板，不再挤在扩展设置页底部。
- 新增关系网络图：中心人物 + 圆形头像节点 + 关系连线 + 关系标签，视觉参考 MOBA 关系图。
- AI 每次回复后，自动在回复结尾追加折叠的「本轮记忆简记」。
- 折叠简记可在面板中关闭，避免聊天正文变长。
- 面板中增加最大输出 token、温度、最多保存简记、关系图节点数量等设置。

## 核心功能

1. 每次 user 和 AI 完成一轮对话后，自动生成一条简略记录。
2. 默认每 20 条简记生成一次阶段总结。
3. 默认 5 个阶段总结合并成一次大总结。
4. 每次生成前自动注入：大总结 + 大总结之后的阶段总结 + 最近简记 + 状态表 + 人物关系表。
5. 实时维护：角色状态、人物档案、人物关系、世界设定、物品栏、约定、当前剧情、发展方向。
6. 自动创建并写入当前角色的记忆世界书。
7. 支持使用 SillyTavern 当前 API，也支持自填 OpenAI-compatible API 地址、Key、模型名。
8. 支持新聊天清空或保留当前角色旧记忆。

## 安装

1. 把本仓库上传到 GitHub，保证 `manifest.json` 在仓库根目录。
2. 打开 SillyTavern。
3. 进入 `Extensions` → `Install Extension`。
4. 粘贴你的 GitHub 仓库地址。
5. 安装后刷新 SillyTavern。
6. 点击右下角 🧠 悬浮球，启用插件。

## 使用建议

### 总结来源

- **使用 SillyTavern 当前 API**：最省事，推荐先用这个。
- **自填 OpenAI-compatible**：适合你想单独用一个便宜模型做记忆总结。填写格式一般是：
  - API 地址：`https://api.xxx.com/v1` 或 `http://127.0.0.1:8000/v1`
  - API Key：你的密钥
  - 模型名：例如 `deepseek-chat`、`gpt-4o-mini`、`Qwen...`

### 折叠简记

开启后，AI 每次回复末尾会自动追加：

```html
<details>
  <summary>🧠 本轮记忆简记 #1</summary>
  ...简记内容...
</details>
```

这会让你在聊天界面直接看到每轮记忆记录。若你不想让聊天正文变长，可以在悬浮面板里关闭「回复结尾折叠简记」。

### 世界书条目

插件会自动创建类似下面的世界书：

```text
RMF-角色名-记忆世界书
```

并写入：

```text
[RMF] 00 当前记忆总览
[RMF] 01 每层简记流水
[RMF] 02 可视化关系表
[RMF] 99 JSON_RAW_DO_NOT_EDIT
```

`JSON_RAW` 用来在「新聊天保留记忆」时恢复状态，不建议手动改。

## 向量化要不要加？

第一版和 0.2.0 都没有做向量化。你的需求更偏「确定性分层总结 + 每次注入总结」，不需要一开始上向量库。

以后聊天特别长、角色很多、需要从几千层前按语义召回某个细节时，再加向量化更合适。当前版本优先保证稳定、直观、可控。

## 注意

- 如果你编辑或删除旧消息，建议点击「补录当前聊天」，让记忆重新生成。
- 浏览器直连部分 API 可能会被 CORS 拦截，这种情况优先使用 SillyTavern 当前 API。
- 本插件会写入当前聊天 metadata 和角色记忆世界书。
